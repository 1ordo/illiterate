## [Unreleased]

### Added
- Initial release of ileterate
- Two-stage grammar pipeline (LanguageTool + LLM)
- Validation loop to prevent LLM hallucinations
- Multi-language support (Dutch, English, German, French, Spanish)
- Cross-platform Flutter app (iOS, Android, Web, macOS, Windows, Linux)
- Real-time grammar error highlighting with animated underlines
- Text rewriting with tone selection (formal, casual, professional, friendly)
- Dark/light theme support
- FastAPI backend with comprehensive error handling
- Docker setup for LanguageTool
- Comprehensive test suite
